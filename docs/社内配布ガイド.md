学宝クイズアプリ 社内配布ガイド
================================

作成日: 2025年12月25日
バージョン: 1.0.0

1. 概要
-------

学宝クイズアプリのテスト版を社内メンバーに配布するための手順書です。


2. ダウンロードリンク
--------------------

2.1 Android版
APKファイル: https://expo.dev/artifacts/eas/9ywHWV6ze7HJRKd8Uy2QnA.apk

インストール方法:
・上記リンクをAndroid端末で開く
・APKファイルをダウンロード
・「提供元不明のアプリ」のインストールを許可（初回のみ）
・ダウンロードしたAPKをタップしてインストール

2.2 iOS版
ビルドページ: https://expo.dev/accounts/panolabo/projects/gakuho-quiz-app/builds/4963de46-1788-42b1-8a82-d8753c9310d2

注意: iOS版はAd Hoc配布のため、事前にデバイス登録が必要です。
新しいiOSデバイスを追加する場合は開発担当者に連絡してください。


3. iOSデバイス追加手順（開発者向け）
----------------------------------

新しいiOSデバイスを追加する場合:

3.1 デバイス登録
ターミナルで以下を実行:
  eas device:create

QRコードまたはURLが表示されるので、追加したいiOSデバイスで開く。
デバイス情報を送信後、登録完了。

3.2 再ビルド
デバイス追加後、アプリを再ビルド:
  eas build --platform ios --profile preview

新しいビルドのリンクを配布対象者に共有。


4. アプリの機能
--------------

・5教科（国語・社会・数学・理科・英語）のクイズ
・制限時間設定（5秒/10秒/30秒/60秒）
・苦手克服モード
・成績統計・弱点分析
・マイページ機能


5. テスト時の確認ポイント
------------------------

・各教科のクイズが正常に動作するか
・タイマーが正しく動作するか
・正解/不正解の判定が正しいか
・結果画面が表示されるか
・マイページの統計が更新されるか
・アプリアイコン（がくまるくん）が表示されるか
・スプラッシュ画面（学宝社ロゴ）が表示されるか


6. 問題報告方法
--------------

バグや改善要望があれば、以下の情報を添えて報告してください:
・端末の種類（iPhone/Android）
・OSバージョン
・発生した問題の詳細
・再現手順
・スクリーンショット（可能であれば）
